CSS - Cascading Style Sheet
===> It is used to style and design a web pages
===> CSS Works together with HTML
===> Change the text, color and Background
===> To set the image and design a layout
===> Adding a Animation and Transition


1) Types of CSS
i) Inline CSS
ii) Internal CSS
iii) External CSS
Inline > Internal > External

2) CSS Syntax
Selector{
    Property:value;
}

3) Types of Selector
i) Elements Selector - (h1),(h2) and (p)
ii) Class Selector - (.) - Multiple - No of times we can use the same class name
iii) Id Selector - (#) - Unique - Only one time we can use the same id name
iv) Universal Selector - (*) - Give some style it will apply into the entire site
v) Group Selector - Class or id name contains different tags, in that different tags we want to change the same style means we can go with Group Selector

6) Box Modeling
i) content
ii) Padding - Around the content is called Padding
iii) Border - Around the Padding is called Border
iv) Margin - Around the Border is called Margin   

7) position
i) Static - (default)
ii) Retative - using position relative to be set the limit like top, bottom, right and left
iii) Absolute - Absolute can search the non-static parent, once reach the non-static parent it will act as a same relative, otherwise it will search one more parent...... there is no non-stactic parent means it will directly go to the root(body) of the website.
iv) Fixed - To be Fixed to the place
v) Sticky - Using top:0 once reach the top:0 it will act as a same fixed.

8) Display - block, inline, inline-block, flex, grid and none

        Flex                                grid
    One dimensional layout          Two dimensional layout
    Row Or Column                     Both Row and Column
    Best for small components        Best for Full page layout 
    Ex: Navbar, cards                  Ex: section, dashbords

========> Flex
i) Using Display:flex (Column into row)
ii) Flex direction - Row, row-reverse, column, column-reverse
iii) Justify content (X) - Start, center, end, Space-around, space-evenly, space-between
iv) Align-items (Y) (set the height) - Start, center, end
v) Gap - to be set the values......... px,rem,cm,m,%,vh,vw
vi) flex wrap - nowrap, wrap, wrap-reverse
vii) Align-content (Y) (set the height) (flex-wrap is enable) - Start, center, end, Space-around, space-evenly, space-between
viii) Flex on item - flex:1,2,3.....
ix) order - Change the visual order of the Property
