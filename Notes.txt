HTML - Hyper text markup language - HTML5   <head> content </head>

Self Closing tag - hr, Meta, br, img, input, link

===> To create a Web pages
===> Not a Programming language
===> Its a  markup language
===> Every website can start with HTML

1) Heading - h1 to h6

2) Paragraph - p

3) Horizontal - hr

4) Break tag - br

5) Text Formatting - u, i, sub, sup, b or strong and mark

6) Anchor Tag (Linking) - a

7) Image tag - img

8) Table - tr, th and td

9) List - ordered list and unordered list

10) Forms - label, input, placeholder, required, button, type(Important), fieldset, legend

11) Media Elements - Audio and video

12) Semantic Tags - Header, footer, section, nav and main

13) Division - Div



CSS - Cascading Style Sheet
===> It is used to style and design a web pages
===> CSS Works together with HTML
===> Change the text, color and Background
===> To set the image and design a layout
===> Adding a Animation and Transition


1) Types of CSS
i) Inline CSS
ii) Internal CSS
iii) External CSS
Inline > Internal > External

2) CSS Syntax
Selector{
    Property:value;
}

3) Types of Selector
i) Elements Selector - (h1),(h2) and (p)
ii) Class Selector - (.) - Multiple - No of times we can use the same class name
iii) Id Selector - (#) - Unique - Only one time we can use the same id name
iv) Universal Selector - (*) - Give some style it will apply into the entire site
v) Group Selector - Class or id name contains different tags, in that different tags we want to change the same style means we can go with Group Selector
